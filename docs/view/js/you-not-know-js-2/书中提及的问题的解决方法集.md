## 竞态模式下的Promise超时模式

```javascript
function timeoutPromise(delay) {
  return new Promise(function (resolve, reject) {
    setTimeout(function () {
      reject('Timeout');
    }, delay);
  });
}
// 三秒超时。
Promise.race([foo(), timeoutPromise(3000)]).then(function () {}，function (err) {
  // foo 被拒绝，或者是超时没完成。
});
```
